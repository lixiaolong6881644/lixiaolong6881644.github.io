一、CentOS7.9 镜像下载

官网下载地址：https://www.centos.org/download/，选择我们准备安装的是版本7面板。

选择我们熟悉的阿里云的镜像地址下载(文件有点大，4个多G)。

四、配置CentOS网络

重启后，让系统自己进入登录页面，中间人工无需任何操作，如下图：

输入root账户和密码，如下图：

使用命令“ping www.baidu.com”, 检查网络是否通，如下图：

网不通，使用命令“ip addr”检查是否有网卡(ens33存在表示有网卡)，如下图：

ens33存在表示有网卡，使用命令“nmcli connection show”,查看网络连接信息（DEVICE栏里是 ‘–’ ,表示没有启用），如下图:

备注：nmcli命令是redhat7或者centos7之后的命令,该命令可以完成网卡上所有的配置工作,并且可以写入配置文件,永久生效。

进入目录/etc/sysconfig/network-scripts/，编辑ifcfg-ens33文件的配置ONBOOT,改成“yes”.重启。
输入 vi ifcfg-ens33，然后看到如下信息，需要注意的是如果是自动获取的话，BOOTPROTO那里要改成dhcp，如果你是静态IP的话，这里要改成static（当然静态IP还要填写IPADDR等信息）。最重要的是要把最后一项ONBOOT=no改成ONBOOT=yes，如下图：

最后输入 “service network restart” 重启网络服务，再次使用命令“nmcli connection show”查看网络连接状况，最后重新“ping www.baidu.com”, 有数据包接收表示配置成功。

五、安装ifconfig命令

使用ifconfig命令查看本机ip, 结果报没有这个命令。

通过命令列出安装包：ifconfig的安装包是net-tools，如下：
$: yum list 文件名

或模糊匹配

$: yum list 文件名*

通过命令安装net-tools安装包，如下：

$: yum install net-tools.x86-64

再次使用ifconfig命令查看ip地址：

此时在ping www.baidu.com,有数据接收 说明虚拟机已经联网
